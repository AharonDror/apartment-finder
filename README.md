# apartment-finder


## Usage
* Create a config.json and put it in the root folder
```
{
  "email": {
    "from": "your-replyto-address",
    "to": ["your-email-address"],
    "sendgridApiKey": "your-sendgrid-api-key"
  }
}
```

* Edit query.js
* Install dependencies:
```
npm install
```
* Run
```
node index.js
```
